BLOG SETUP

  •  

  •  

There are 7 kind of blog posts:

  • 1: General Post
  • 2: Slider Post
  • 3: Audio Post
  • 4: Youtube Video Post
  • 5: Vimeo Video Post
  • 6: MP4 Video Post
  • 7: Quote Post

 

 

 

1: general post

 

Share Button should have an unique id, and this id should have reffer to the share dropdown toptions by for. Please see the code for more clearness. this is importent for every kind of post.

 

 

 

 

 

2: slider POST

 

Provide all of images in the Thumb-slides-container div.

 

 

 

 

 

3: audio POST

 

For Every Audio posts should have unique ID like audio1 and please provide the audio file link in the data-file-src. and you should give the audio-post class in the article tag.

 

 

 

 

 

4: youtube video POST

 

Past your youtube video inside the embed-responsive-item div.

 

 

 

 

 

5: vimeo VIDEO POST

 

Past your vimeo video inside the embed-responsive-item div.

 

 

 

 

 

6: mp4 VIDEO POST

 

For Every MP4 Video posts should have unique ID like video1. and please provide the audio file link in the data-file-src. and give the video thumbnail in the data-image-src.

 

 

 

 

 

7: quote POST

 

Past your quote in the "Pur your awesome ...". and you should give the quote-post class in the article tag.

GOOGLE MAP

 

 

Create Google Map API Key & Place in teh HTML

 

At first you need a Google Map API key to place in the below image highlighted area "PROVIDE_YOUR_API_KEY".

how you can get the API key? just click here: Get a key. it will automatically create a default project, select api key type as browser key, and show you the Credentials screen like below to create the API key.

Then you have to add your web sites url. if your website is "http://example.com" then provide "http://example.com/*". so API can only accept all of the requests from your whole site. if you don't understand what is this you can simply leave it.

Now click on Create button, then it you give you a popup or a page like below, where you see the api key

If you need to learn more about it, just visit Here

 

Config you map location & others

In the personx/assets/js/main.js file, change the lat , lng , content. as your wish, you can also change the marker icon by providing the icon url in the icon option.
